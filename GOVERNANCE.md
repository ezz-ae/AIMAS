# Governance

## Canonical authority
This repository defines the AIMAS protocol. RFC-0007 is binding.

## Change policy
- Backward-incompatible schema changes require a new major version.
- Any change that weakens RFC-0007 invariants is rejected.

## Attestation (recommended)
Implementations should publish:
- commit hash of this repo
- conformance test results
- declared E0/E1 TTL policy
