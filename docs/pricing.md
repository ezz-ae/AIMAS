| Tier | Description | Included |

All tiers include:
- Deterministic Fit Matrix computation
- Cloud Run deployment template + probes
- Access to AIMAS conformance suites

Billing is subscription-based. Buyers pay to remove unfairness risk, not to buy throughput.
